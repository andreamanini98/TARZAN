Computes whether a network region with locations equal to targetLocs is reachable from the set of initial network regions.

Parameters
targetLocsthe locations of the network region that must be reached.
explorationTechniquedetermines the state space exploration technique (e.g., BFS, DFS).
Returns
a vector containing the target network region if it is reachable, an empty vector otherwise.
Warning
The size of targetLocs must match the number of automata in the network. Use std::nullopt to declare a value as missing from the reachability evaluation.

Computes whether a network region with locations equal to targetLocs is reachable from the set of initial network regions.

Parameters
intVarConstra vector of clock constraints that act as integer variables constraints. Instead of a clock, each clock constraint contains the name of an integer variable.
goalClockConstraintsconstraints that must hold for the goal network region to be reachable.
targetLocsthe locations of the network region that must be reached.
explorationTechniquedetermines the state space exploration technique (e.g., BFS, DFS).
Returns
a vector containing the target network region if it is reachable, an empty vector otherwise.
Warning
The size of targetLocs must match the number of automata in the network. Use std::nullopt to declare a value as missing from the reachability evaluation.
The size of goalClockConstraints must match the number of automata in the network. Use an empty vector to declare a value as missing from the reachability evaluation.
The intVarConstr is a vector of clock constraints, although here, instead of clocks, integer variables must be used.
