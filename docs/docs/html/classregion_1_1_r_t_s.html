Computes whether a region with a location equal to targetLocation is reachable from the set of initial regions.

Parameters
intVarConstra vector of clock constraints that act as integer variables constraints. Instead of a clock, each clock constraint contains the name of an integer variable.
goalClockConstraintsconstraints that must hold for the goal region to be reachable.
targetLocationthe location of the region that must be reached.
explorationTechniquedetermines the state space exploration technique (e.g., BFS, DFS).
Returns
a vector containing the target region if it is reachable, an empty vector otherwise.
Warning
The intVarConstr is a vector of clock constraints, although here, instead of clocks, integer variables must be used.
