Computes the satisfiability of an invariant.

Parameters
invarianta vector of clock constraint whose satisfiability must be computed.
clockValuationthe current clock valuation (integer values and a boolean denoting whether the fractional part is greater than zero).
clocksIndicesthe indices of the clocks as they appear in the clocks vector of a Timed Automaton.
Returns
true if the guard is satisfied, false otherwise.
Attention
Works only if the guard is a conjunction of clock constraints, where a clock constraint is (x ~ c), with ~ in {<, <=, =, >=, >}.
