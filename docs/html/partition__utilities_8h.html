◆ partitionBitset()

void partitionBitset ( const boost::dynamic_bitset<> & bitset,
std::vector< std::vector< int > > & partitions,
std::vector< int > & activeBitsIndices )
inline

The implementation follows the paper: Maximize the Rightmost Digit: Gray Codes for Restricted Growth Strings.

A partition is a Restricted Growth String (or Bell String), denoting the set in which elements must belong. Elements are represented by their integer index, since we only want to partition elements in the bitset. For example, if we have bitset = 10010, then the elements to partition correspond to indices 0 and 3.

Parameters
bitsetthe bitset to partition (only partitions the bits set to 1).
partitionsa std::vector in which to store the partitions of bitset.
activeBitsIndicesa std::vector in which to store the indices of the elements that have been partitioned.
